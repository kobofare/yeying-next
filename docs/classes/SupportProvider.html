Class SupportProvider

  • collectFaq(type: string, email: string, description: string): Promise<void>

  • 收集 FAQ 数据并发送到支持服务。

    Parameters

    • type: string

      FAQ 类型，例如“general”。

    • email: string

      提交 FAQ 请求的电子邮件地址。

    • description: string

      FAQ 问题的描述。

    Returns Promise<void>

    一个 Promise，解析为 void，表示请求成功。

    Throws

    如果签名或请求头创建失败，则抛出错误。

    Throws

    如果协议错误或响应失败，则抛出错误。

    Example
