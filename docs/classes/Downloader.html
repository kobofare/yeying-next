Class Downloader

Downloader 类用于下载资产数据，支持从区块提供者获取数据并进行解密。

Example

const downloader = new Downloader(blockProvider, assetCipher);
downloader.download(asset).then(blob => {
  // 下载成功后处理数据
}).catch(err => {
  // 处理下载错误
});
